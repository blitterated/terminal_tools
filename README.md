# Terminal Tools

This is a SQLite database I use to track all the terminal tools I use or am interested in exploring.

| Script                    | Description      |
| ------------------------- | ---------------- |
| `boilerplate.sql`         | Contains boilerplate SQL for inserting new tools. |
| `dump.sh`                 | Dumps the schema and data to `terminal_tools_dump.sql`. |
| `restore.sh`              | Restores the schema and data from `terminal_tools_dump.sql`. |
| `scratch.sql`             | Scratchpad for running queries. Has some useful queries already. |
| `terminal_tools_dump.sql` | Holds the SQL dump of both schema and data. |
